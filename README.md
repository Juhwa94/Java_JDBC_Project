# 영화 티켓 관리 프로그램
상영 영화 정보와 영화 티켓 정보를 관리할 수 있도록 제작한 콘솔 기반 Java 프로그램 입니다.
## 👨‍🏫 프로젝트 소개
사용자는 영화 등록, 조회, 수정, 삭제 기능과 함께 티켓 등록, 조회, 수정, 삭제 기능을 사용할 수 있습니다. 영화와 티켓은 1:N 관계로 구성되어 있어 하나의 영화에 대해 여러 자의 티켓 정보를 관리할 수 있습니다. 레이어드 아키텍처를 적용하여 유지보수성과 확장성을 높였습니다. 효율적으로 상영 영화의 정보를 제공하고 수정하고 삭제하기 위하여 제작하게 되었습니다.

## 📌 주요 기능
- 영화 등록 : 사용자가 제목, 장르, 감독, 상영시간 등의 영화 정보를 입력하여 등록한다.
- 영화 조회 : 데이터베이스에 저장된 영화 목록을 조회하고 상세 정보를 확인한다.
- 영화 수정 : 기존에 등록된 영화 정보를 선택하여 원하는 내용으로 변경한다.
- 영화 삭제 : 선택한 영화 정보를 데이터베이스에서 삭제한다.
