# 영화 티켓 관리 프로그램
상영 영화 정보와 영화 티켓 정보를 관리할 수 있도록 제작한 콘솔 기반 Java 프로그램 입니다.
## 👨‍🏫 프로젝트 소개
사용자는 영화 정보와 티켓 정보에 대해 등록(Create), 조회(Read), 수정(Update), 삭제(Delete) 기능을 수행할 수 있습니다. 영화와 티켓은 1 관계로 설계하여 하나의 영화에 여러 티켓 정보를 등록하고 관리할 수 있도록 구현하였습니다. 또한 Layered Architecture를 적용하여 프레젠테이션 계층, 서비스 계층, 데이터 접근 계층을 분리함으로써 유지보수성과 확장성을 높였습니다. 사용자 입력은 Front에서 받고, Service에서 비즈니스 로직을 처리하며, DAO에서 JDBC를 통해 DB와 통신합니다. 데이터는 VO 객체를 통해 각 계층 간 전달되도록 설계했습니다.

## 📌 주요 기능

🎬 영화 관리
- 영화 등록 : 제목, 장르, 감독, 상영시간 등의 영화 정보를 등록
- 영화 조회 : 등록된 영화 목록 및 상세 정보 조회
- 영화 수정 : 기존 영화 정보 수정
- 영화 삭제 : 영화 정보 삭제

🎫 티켓 관리
- 티켓 등록 : 영화별 예매 티켓 정보 등록
- 티켓 조회 : 영화에 등록된 티켓 정보 조회
- 티켓 수정 : 기존 티켓 정보 수정
- 티켓 삭제 : 티켓 정보 삭제
